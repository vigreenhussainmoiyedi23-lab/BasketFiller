# Basket Filler

**Basket Filler** is a full-stack e-commerce platform designed to provide a seamless shopping experience for users while allowing admins to manage products, orders, and analytics efficiently. The project is built with **Node.js, Express, MongoDB, and React**.

## Features

### User Features
- **User Authentication**: Register, login, logout with JWT & cookies.  
- **Profile Management**: Update profile details.  
- **Product Browsing**: View all products with search, filter, and pagination.  
- **Cart Management**: Add, remove, and update products in cart.  
- **Order Management**: Place orders and view order history.  
- **Product Recommendations**: Shows recommended products based on past orders or cart items.  
- **Comments & Reviews**: Users can leave comments and reviews on products.  
- **Responsive UI**: Works seamlessly across mobile, tablet, and desktop.  

### Admin Features
- **Admin Authentication**: Secure login for admins.  
- **Product Management**: Add, edit, delete products.  
- **Order Analytics**: View revenue statistics by day, month, year.  
- **User Management**: View registered users and their activity.  
- **Comment Moderation**: Manage product comments and reviews.  
- **Graphs & Analytics**: Product sales graphs and revenue visualizations.  

---

## Tech Stack
- **Backend**: Node.js, Express  
- **Frontend**: React.js  
- **Database**: MongoDB (Mongoose)  
- **Authentication**: JWT + Cookies  
- **Other Libraries**: Axios, CORS, cookie-parser, bcrypt, etc.  
- **Deployment**: Vercel (Frontend), Render (Backend)
